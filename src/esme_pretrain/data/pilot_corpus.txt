Small models learn from small, honest corpora.
The pilot corpus is committed text, not a remote download.
Every line is plain enough for the first tokenizer path to inspect.
Raw text becomes tokens, tokens become packed training windows.
The model sees short sequences on CPU and learns the next character.
Loss must move down before bigger data earns the next milestone.
Checkpoint reload must reproduce fixed logits exactly.
Samples are allowed to be awkward; silent failures are not.
This repo proves the base-model path before any paid compute appears.
Pilot runs stay compact, deterministic, and local.
